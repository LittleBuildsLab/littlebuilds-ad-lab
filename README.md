# LittleBuilds AD Lab

A hands-on Windows Server Active Directory homelab project for practicing real-world IT administration skills.

## Project Goal

The goal of this lab is to build a small business-style Windows domain using Windows Server and a Windows client VM. This project will document the setup process, configuration choices, screenshots, troubleshooting notes, and lessons learned.

## Planned Lab Environment

- Virtualization platform: Oracle VirtualBox
- Server OS: Windows Server 2022
- Client OS: Windows 11
- Domain concept: LittleBuildsLab
- Focus areas:
  - Active Directory Domain Services
  - Users and groups
  - Organizational Units
  - Shared folders and permissions
  - Group Policy
  - DNS basics
  - PowerShell practice

## Planned Project Sections

- Server installation
- Domain controller setup
- Active Directory structure
- User and group creation
- Shared folder permissions
- Group Policy testing
- Windows client domain join
- Troubleshooting notes
- Screenshots and documentation

## Why I Built This

I am building this lab to strengthen my practical IT skills and create a documented portfolio project that shows hands-on experience with Windows Server, Active Directory, virtualization, and troubleshooting.
